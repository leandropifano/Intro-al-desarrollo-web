Posicionamiento por DEFECTO 03
Posicionamiento ABSOLUTO
top:50% left:50%

El valor absolute en el atributo position permite posicionar elementos de manera absoluta, esto es de manera definida por valores de los atributos top, left, bottom y right, que indican la distancia con respecto a un punto. Esto quiere decir que no importa donde este el ultimo elemento insertado ya que el posicionamiento dependera solo de la pantalla y los valores de los atributos.

Posicionamiento FLOTANTE
"Izquierda" 01

La propiedad float establece el esquema de posicionamiento flotante para un elemento. Cuando existe un elemento flotante, los elementos que se encuentran a continuación del elemento flotante fluyen a lo largo de él, salvo que haya un elemento que tenga establecido la propiedad clear. Las propiedades float y clear se pueden aplicar a cualquier elemento de una página web.

Posicionamiento FIJO

Este atributo sirve para posicionar una capa con posicionamiento absoluto, pero su posición final será siempre fija, es decir, aunque se desplace el documento con las barras de desplazamiento del navegador, siempre aparecerá en la misma posición.
