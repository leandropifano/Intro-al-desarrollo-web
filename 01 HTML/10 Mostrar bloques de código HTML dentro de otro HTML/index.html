Mostrar bloques de código HTML dentro de otro HTML

Para mostrar bloques de código HTML dentro de un documento HTML sin que sea interpretado por el navegador las etiquetas deberan respetar la siguiente regla:
En lugar de escribir la etiqueta de esta forma '<'ETIQUETA'>' (sin comillas)
Deberá ser escrita de esta otra forma: &lt;ETIQUETA&gt;


Ejemplo:
<body>
